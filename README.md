# Hunt360 Platform

## Setup Instructions

1. **Assets**: Place your images in `client/public/assets/`.
   - Rename the logo to `logo.png`.
   - Rename the world map to `world-map.png`.
2. **Installation**: Run `npm run install-all` in the root directory.
3. **Environment**: Update `server/.env` with your email credentials for MFA.
4. **Run**: Execute `npm start` to run both frontend and backend.

## Features
- MFA (Email OTP)
- Password Strength Enforcement
- Dashboard Analytics
